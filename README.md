# 바이브코딩 실습 프로젝트 (파이어베이스 연동)

이 저장소는 유튜브 조코딩 채널의 바이브코딩 강의를 보며 실습하기 위해 구성된 저장소입니다.  
현재 파이어베이스(Firebase) 프로젝트로 인식되게끔 설정 파일들이 포함되어 있으며, 간단한 **로또 번호 추첨기** 웹 애플리케이션이 완성되어 있습니다.

## 주요 기능
- **로또 번호 추첨기**: 1~45 중 랜덤하게 당첨 번호를 추첨하여 애니메이션 효과와 함께 화면에 렌더링합니다.

## 디렉토리 구조
```text
vibe_coding/
├── .firebaserc       # 파이어베이스 프로젝트 설정(프로젝트 아이디 등)
├── firebase.json     # 파이어베이스 호스팅 및 배포 설정
├── index.html        # 메인 HTML 페이지 
├── style.css         # UI와 애니메이션 스타일 
├── script.js         # 로또 추첨 로직 및 DOM 조작 
└── README.md
```

## 실행 방법
- 현재 폴더 안의 `index.html` 파일을 웹 브라우저에서 직접 더블클릭해서 열어보실 수 있습니다.
- VS Code 라이브 서버(Live Server) 확장 프로그램이나, 다른 로컬 웹 서버로 열면 더욱 쾌적하게 실습할 수 있습니다.

## 파이어베이스 연동 및 배포
- 터미널에서 `firebase login`으로 로그인한 후 파이어베이스 프로젝트를 연결하세요.
- `firebase deploy` 명령어를 입력하면 `firebase.json` 설정에 따라 현재 폴더의 파일들이 파이어베이스 호스팅을 통해 배포됩니다.
